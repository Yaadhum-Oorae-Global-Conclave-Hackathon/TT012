மைக்ரோஃபோன் ஐகானைக் கிளிக் செய்து, நீங்கள் விரும்பும் வரை பேசத் தொடங்குங்கள்!

பேசத் தொடங்குங்கள்!

பேச்சு எதுவும் கண்டறியப்படவில்லை. உங்கள் அமைப்புகளை நீங்கள் சரிசெய்ய வேண்டியிருக்கலாம்.

மைக்ரோஃபோன் எதுவும் கிடைக்கவில்லை. மைக்ரோஃபோன் நிறுவப்பட்டுள்ளதா என்பதை உறுதிப்படுத்தவும் மைக்ரோஃபோன் அமைப்புகள் சரியாக உள்ளமைக்கப்பட்டுள்ளன.

உங்கள் மைக்ரோஃபோனை இயக்க மேலே உள்ள "அனுமதி" பொத்தானைக் கிளிக் செய்க.

மைக்ரோஃபோனைப் பயன்படுத்த அனுமதி மறுக்கப்பட்டது!

மைக்ரோஃபோனைப் பயன்படுத்துவதற்கான அனுமதி தடுக்கப்பட்டுள்ளது. மாற்ற, செல்லுங்கள் chrome://settings/contentExceptions#media-stream

வலை உரை API ஐ இந்த உலாவி ஆதரிக்கவில்லை. Chrome பதிப்பு 25 அல்லது அதற்குப் பிறகு மேம்படுத்தவும்.

உரையை நகலெடுக்க கண்ட்ரோல்-சி அழுத்தவும்.

(மேக்கில் கட்டளை-சி.)
